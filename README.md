
1. What is the difference between var, let, and const? 

In JavaScript, var, let, and const are used to declare variables. Their main differences are scope, re-declaration, and reassignment.

a. var is function scoped. It can be re-declared and reassigned. It is hoisted and initialized with undefined.

b. let is block scoped. It cannot be re-declared in the same scope. It can be reassigned.

c. const is also block scoped. It cannot be re-declared or reassigned. The value must be assigned at the time of declaration.

2.  What is the spread operator (...)?

 The spread operator (...) in JavaScript is used to expand or spread elements of an array or object into individual elements. It can copy or merge array and objects.

 3. What is the difference between map(), filter(), and forEach()?
 These are array methods used to loop through an array.

 a. map(): It returns a new array with modified values and the original array dose't change.

 b. filter(): It's Used to select elements based on a condition. It returns a new array with elements that pass the condition.

 c. forEach(): Used to loop through an array. It does not return a new array. Mostly it's used to updating values.

 Template literals are a way to write strings in JavaScript. It is using backticks (``) instead of single (' ') or double (" ") quotes.
